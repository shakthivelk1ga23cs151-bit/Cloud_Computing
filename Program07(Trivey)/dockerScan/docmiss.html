Misconfigurations

app/Dockerfile

ID Severity Title Description
DS-0002 HIGH Image user should not be 'root' Running containers with 'root' user can lead to a container escape situation. It is a best practice to run containers as non-root users, which can be done by adding a 'USER' statement to the Dockerfile.
DS-0026 LOW No HEALTHCHECK defined You should add HEALTHCHECK instruction in your docker container images to perform the health check on running containers.
